# Strategy Publisher Public Blog

Static GitHub Pages-ready publication for Auto Company's strategy writing.

## Purpose

This directory is the local source artifact for Nick's requested public blog/newsletter. It is designed to be copied or published as a GitHub Pages source directory later, but the local cycle does not deploy, push, configure Pages, or use external accounts.
